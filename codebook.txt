This codebook corresponds to tidy.data dataframe which is a long format dataset that summarise data in the Human Activity Recognition Using Smartphones Dataset.
The treatment of Human Activity Dataset is described in the README file and the run_analysis script contain the managing of the data.
Column	Variable	Description
1	id		Subject Id
2	activity	Activitiy for which the measuremen was made.
3	varible		Indicate for which variable the mean measure was computed
4	mean		Mean for the variable of column 3 for the activity and subject corresponding to columns 2 and 1 respectively
